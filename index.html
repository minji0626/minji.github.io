About Myself

웹 개발에 대한 열정과 끊임없는 배움을 추구하는 신입 개발자입니다. 대학에서 웹 프로그래밍 수업을 통해 HTML, CSS, JavaScript 등 다양한 웹 기술을 익히며 웹 개발에 대한 깊은 흥미를 발견했습니다. 사용자와 상호작용하는 웹 페이지를 직접 구현하면서 사용자 경험을 개선하기 위해 고민하는 과정에서 웹 개발의 매력에 빠졌고, 이를 바탕으로 웹 개발자로서의 경로를 선택하게 되었습니다. 졸업 후 쌍용 교육센터에서 추가적인 교육을 받으며 실무 중심의 프로젝트를 통해 실력을 쌓아왔습니다. 이 과정에서 팀 프로젝트를 통해 협업의 중요성을 깨닫고, 문제 해결 능력을 키우며 빠르게 성장할 수 있었습니다. 웹 개발은 계속해서 발전하는 분야이기에, 최신 기술과 트렌드에 맞춰 지속적으로 학습하고 성장할 수 있는 기회를 갖는 것이 저에게 큰 동기부여가 됩니다. 팀워크와 책임감을 중요하게 생각합니다. 저는 규칙을 잘 준수하며, 필요할 때 리더십을 발휘하는 동시에 팀의 일원으로서 팀을 지원하는 것을 중요시합니다. 문제 해결 과정에서 적극적으로 참여하고, 다른 팀원들과 지식을 공유하며 함께 성장하는 것을 좋아합니다. 항상 성실하게 임무를 수행하며, 팀의 목표를 달성하기 위해 최선을 다합니다. 앞으로 저는 풀스택 개발자로서 프론트엔드와 백엔드 양측에서 사용자에게 안정적인 웹 서비스를 개발하고자 합니다. 최신 웹 기술을 꾸준히 습득하며, 팀원들과의 원활한 소통과 협업을 통해 프로젝트를 성공적으로 이끌고 싶습니다. 회사의 목표와 가치에 부합하는 책임감 있는 자세로 업무를 수행하며, 성실하게 회사의 성장에 기여하는 개발자가 되겠습니다.
